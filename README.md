# Научиться учиться

Первый проект на курсе Веб-разработчик в [Яндекс.Практикуме](https://practicum.yandex.ru/).
Вёрстка страницы с использованием семантики, БЭМ, флексбоксов, гридов и анимаций.

## Стек

* HTML
* CSS: flexbox, grid, animations
* BEM
